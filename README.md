# photo-frame-epaper
Firmware for the epaper photo frame project

## Requirement
- PlatformIO
- C Toolchain

## Hardware
- ESP32 Dev board
- Waveshare 5.56in E-Ink display
- DS3231 RTC
- TP4056 Charging board
- Lipo cell
- MCP1700 LDO

## Build and Flash
Download VSCode, install PlatformIO plugin, select target, connect your board to computer, and hit flash.
